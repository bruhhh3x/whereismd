# Access Initiation (React)

A mobile-first, timed access initiation experience.

## Features
- 8-second instinct window per question
- Randomized question pool
- Touch-friendly UI
- Designed for cultural and experiential gating

## Setup
Recommended with Vite:

```bash
npm create vite@latest access-initiation -- --template react
cd access-initiation
npm install
npm run dev
```

Replace `src/App.jsx` with the included file.
